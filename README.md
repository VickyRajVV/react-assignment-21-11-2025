# React Assignment 21-11-25


## Quick start (VS Code)


1. Create project folder and paste files from this document.
2. Run `npm install` to install dependencies declared in package.json.
3. Start dev server: `npm run dev`.
4. Open the URL printed by Vite (usually http://localhost:5173).


Notes:
- This project uses functional components, hooks (useState, useEffect, useRef, useMemo, useReducer) and a small reducer-based fetch hook.
- API: https://jsonplaceholder.typicode.com/


Files of interest: `src/components/*` and `src/hooks/useFetchReducer.js`.
